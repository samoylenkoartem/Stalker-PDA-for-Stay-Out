# Stalker-PDA-for-Stay-Out
🗺️ Stalker PDA for Stay Out
AI-компаньон для онлайн-игры Stay Out — автоматически отслеживает игровые события в реальном времени и генерирует тактические отчёты.
Что умеет

📊 Отслеживает HP через анализ пикселей экрана
💾 Логирует ранения и лечение в базу данных SQLite
🔴 Black Box режим — повышает частоту опроса при критическом HP
🤖 Генерирует AI-отчёт по итогам сессии через Groq/Llama
🪟 Динамически привязывается к окну игры

Стек

Python
SQLite3
pytesseract
pygetwindow
pyautogui
Groq API (Llama 3.1)

Статус
🚧 В активной разработке — проект документируется в формате dev-vlog на YouTube.
